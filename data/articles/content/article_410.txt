Features This module provides integration between Drupal and Memcached with the following features: An API for using Memcached and the PECL Memcache or Memcached libraries with Drupal. Memcache backends for the following systems (all drop-in): Caching (memcache.inc) Locking (memcache-lock.inc) Sessions (memcache-session.inc) (D6 only) A module that provides a comprehensive administrative overview of Drupal's interaction with Memcached and stats.
